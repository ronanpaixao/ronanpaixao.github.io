Lançado PirateShip 1.1

Date dom 25 abril 2010 Tags download / linux / PirateShip / programa

PirateShip
Hoje foi lançado a segunda versão do PirateShip, com diversos melhoramentos, que foram pedidos dos usuários. Os mais importantes:

  • Possibilidade de inverter as cores dos cabos, pois algumas pessoas preferem conectar o cabo colorido em outra direção;
  • Uma primeira versão de um sistema rudimentar de plugins. Como primeiro plugin, o PirateShip na versão compilada dinamicamente já inclui um plugin para interfacear com um magnetômetro HMC5843;
  • Uma versão compilada para Linux!

Faça já o download da nova versão na página do PirateShip!
