Longo tempo fora, mas não parado!

Date ter 02 novembro 2010 Tags chavant / PFC / Projeto Final de Conclusão / retorno

Olá. Estive algum tempo sem atualizar o site, devido a falta de tempo, mas agora deve dar uma melhorada.

O motivo de estar sem tempo é que estava me dedicando à conclusão do curso de Engenharia Eletrônica, assim como ao meu Projeto Final de Conclusão. O meu projeto foi realizar a construção de uma UMI (Unidade de Medida Inercial, ou IMU, de Inertial Measurement Unit).

Este foi um projeto bastante trabalhoso, que estou desenvolvendo desde o ano passado. Ele inclui não apenas o hardware para medidas, mas também um microcontrolador capaz de unir os dados dos sensores e processá-los, transformando a plataforma em um AHRS (Attitude and Heading Reference System), que é um sistema capaz de indicar a orientação da plataforma. Este é o primeiro passo para a construção de um piloto automático para o Projeto VANT.

A partir dos dados deste projeto, aperfeiçoaremos os algortimos e o hardware, em busca de um piloto automático efetivamente brasileiro.
